# 指南 {#指南}

## 为什么需要它? {#为什么需要它}

* 它能生成整个项目API代码，interface，让开发者脱离复制API代码、定义interface的苦劳，更多精力关注于业务代码即可，一次生成整个团队共享。
* 模板代码带来的团队规范性，模板代码的生成，让团队成员核心业务代码流程一致，带来更好的维护性以及可读性，看团队成员代码就像看自己代码一样。
