# [API Helper](https://github.com/ztz2/api-helper)

`API Helper` 是一个代码生成工具，可根据 [Swagger](https://swagger.io/) 的接口定义生成 TypeScript 或 JavaScript 的接口类型及其请求函数、类、对象代码，以及根据特定API请求参数和响应参数生成模板代码。
